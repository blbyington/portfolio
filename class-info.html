ABOUT THE CLASS

Topics include:

  • Proper use of HTML5, including new semantic elements
  • Custom styling including web fonts
  • Introduction to client server architecture
  • Principles of effective user experience
  • Media playback with new HTML5 media elements
  • Introduction to Bootstrap

Course Learning Objectives:

  1. Strong understanding of HTML and CSS fundamentals
  2. Ability to write semantic, valid, and reusable HTML and CSS
  3. Ability to translate design mockups into working code
  4. Understanding of web accessibility
  5. Functional understanding of modern version control
